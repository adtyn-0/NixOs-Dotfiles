# NixOS Config & Dotfiles

Personal NixOS configuration and dotfiles.

Built for my daily workflow using:

- Modular NixOS configuration
- Hyprland, Niri, KDE (depending on mood)
- Neovim (custom Lua setup)
- Waybar, Kitty/Ghostty


## Structure

- `nixos/` → system configuration (modules, hardware, services)
- `configs/` → user configs (nvim, hyprland, waybar, etc.)

## Notes

- This setup is tailored to my hardware and preferences
- Not intended to be plug-and-play
- You will have to change things. That’s the point :)

## Philosophy

If it improves my daily workflow, it stays.
